# Agent Tools & Interoperability Website

This is a static website presenting excerpts from the Agent Tools & Interoperability whitepaper.

## Features
- Clean responsive design
- Whitepaper content highlighted
- Random filler words for stylistic variety

## How to Run
Open `index.html` in your browser.

## Deployment
1. Push this repo to GitHub.
2. Go to **Settings > Pages**.
3. Select branch `main` and folder `/root`.
4. Your site will be live at `https://yourusername.github.io/agent-tools-website` e.g .
